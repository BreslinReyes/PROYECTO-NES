Acerca de Nuestro Prototipo

El prototipo sería algo parecido a esto, pero sin la pulsera, esto iría adherido a los postes de luz para cumplir su función, tendría un medidor de decibeles interno e iría conectado a una base de datos donde enviara los datos recolectados.

Componentes

Transciver

Este es el que se encarga de transmitir la información a la antena, la cual la recopilara en una base de Datos.

Para Rayos

El pararrayos será de cobre para evitar que se dañe el dispositivo con una caída de un rayo.

Medidor de decibeles

El medidor de decibeles tendrá un límite establecido, que sería el límite legal, el dispositivo estará Conectado a través de la red global con una velocidad de 2G.

Materiales de el Prototipo

Estos los que componen el dispositivo como:

  • -Sensor de movimiento
  • -Plasticos
  • -Cableados
  • -Pantalla
  • -Soporte

Proceso de Dispositivo NES

Como Funciona el dispositivo: El dispositivo se coloca en un poste de luz, este contendrá un medidor de decibeles el cual tendrá un límite establecido, que sería el límite legal, el dispositivo estará conectado a través de la red global con una velocidad de 2G para que este pueda enviar la información que vaya recolectando a una base de datos de la policía, el poste de luz tendrá un para rayos de cobre para evitar que se dañen el dispositivo con una caída de un rayo.
